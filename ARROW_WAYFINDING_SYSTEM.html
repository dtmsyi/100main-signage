Arrow Wayfinding System - Delivery Path Signage

Complete Delivery Path Journey

Purpose: Create a clear, unambiguous path from the lobby entrance to the courier door using progressive arrow signage.

Step 1: Lobby Entrance
Primary sign above intercom
"Do not use intercom"
→
Step 2: Exit & Turn
Ground decal arrow
"Deliveries this way →"
→
Step 3: Around Corner
Wall-mounted arrow
"Package room →"
→
Step 4: At Door
Final directional sign
"Courier entrance ←"

Design Principle: Progressive disclosure - each sign reinforces the message and guides drivers step-by-step to the correct entrance.

Complete System Summary

Total Signage Components:

  • Primary Sign: 12" × 8" above Butterfly intercom (24 design options available)
  • Arrow 1: 10" × 10" exit/turn right (6 color options)
  • Arrow 2: 10" × 10" around corner (6 color options)
  • Arrow 3: 10" × 12" courier entrance left (6 color options)
  • Ground Decal (Optional): 18" × 6" arrow (3 color options shown)

Material Requirements for ALL Signs:

  • Outdoor-grade, weather-resistant (no curling/warping)
  • UV-protected for long-term exposure
  • Reflective preferred for nighttime visibility
  • Consistent color scheme across all signs

Installation Approach:

  • Phase 1: Install primary sign above intercom + ground decal
  • Phase 2: Monitor effectiveness for 1 week
  • Phase 3 (if needed): Add supplementary arrow signs along pathway
